# Create Verifiable Credential and Verifiable Presentation using Dock SDK

## How to use

1. Install dependencies
```bash
yarn install
```

2. Start development server
```bash
yarn start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
